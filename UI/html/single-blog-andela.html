What is Andela and why did I join the Andela technical leadership program?

07 December. 2021

Andela Technical Leadership Program (ATLP) is a nine month program implemented by Andela in partnership with the Ministry of ICT, with the mission to empower and equip Rwandese with required skills in software engineering to become world-class engineers. The program also provides professional skills to help them to be able to work in any work environment. After getting informed about the program and how it empowers young Rwandese nationals to become world-class engineers, I was very excited to apply for the second cohort. I applied and did the technical assessment they gave me but I failed. I did not give up. I kept my hope to be part of the program in the next cohort, in the fourth cohort I applied again, and finally, my dream came true I succeeded. So far in the Bootcamp, I have gained skills in web development using technologies like HTML5, CSS3, javaScript, firebase, and node js. We have skilled and experienced mentors who help us in our daily learning progress by making sure that we are meeting the expectations of ATLP. Another thing I have learned so far in the program is teamwork and collaboration, we help one another to be on the same standard. In the professional skills, I have learned the importance of personal branding and how to brand myself.
